# MINDFLAG Project

This is a country/flag app powered by MINDBODY, INC.

## Installation

Run on Xcode 11 (or higher)

## Key Features
- Uses Codable protocol to parse JSON
- Uses MapKit to display countries/provinces
- Support for Dark Mode 
- Compatible with iPad
